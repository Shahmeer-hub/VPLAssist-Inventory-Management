# VPLAssist+ Inventory Management System

## Description
VPLAssist+ is a desktop-based Inventory Management System developed using C# and WPF.
The application allows users to add, update, delete, and view product records through a visual interface.
Data is stored locally using a JSON file.

This project was developed as a Complex Computing Problem (CCP) for the VPL (Theory) course.

## Features
- Add, update, delete, and view products
- DataGrid-based record display
- Product search functionality
- Inventory summary (total products, total value)
- JSON file data persistence
- Basic multithreading using async/await
- Error handling with validation and try-catch

## Technologies Used
- C#
- WPF (.NET)
- JSON (Newtonsoft.Json)

## How to Run
1. Open the project in Visual Studio
2. Restore NuGet packages if required
3. Run the project using the Start button
